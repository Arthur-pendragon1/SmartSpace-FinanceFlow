# SmartSpace & FinanceFlow

Proyek ini berisi dua tampilan frontend sederhana yang siap diunggah ke GitHub:

- SmartSpace: landing page dan dashboard coworking space
- FinanceFlow: dashboard manajemen keuangan pribadi dengan modal transaksi

## Struktur Folder

- client 1/ -> SmartSpace
- client 2/ -> FinanceFlow

## Cara Menjalankan Lokal

Masuk ke folder masing-masing lalu jalankan:

```bash
npx http-server
```

Lalu buka:

- http://localhost:8080

## Upload ke GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
gh repo clone Arthur-pendragon1/SmartSpace-FinanceFlow
git push -u origin main
```
